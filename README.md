# React Native Ground

> React-Native style을 연습해보기 위한 레포지토리입니다.
> [unsplash](https://unsplash.com/) image를 사용했습니다.

## Installation

```
git clone https://github.com/NeedSilentPlace/purple-ground.git
cd purplestore_front
npm i
cd ios
pod install
cd ..
npm start
npm run ios
```

## Structure

- public
  - **lottie**: lottie json file
- src
  - navigation
  - ui
    - **unit**
    - **component**
    - **screen**
    - **color**
